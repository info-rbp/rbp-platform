## Review Rules

Before external material is used in the application:

1. Confirm source and ownership.
2. Confirm whether it can be committed.
3. Convert raw content into app-safe copy or mock data.
4. Identify legal review needs.
5. Identify Phase 2 backend relevance.
6. Record usage in this register.

## Do Not Commit Without Review

- Raw client documents
- Raw exports
- Large binary files
- Payment details
- Personal data
- Legal copy requiring review
- Production credentials
- API keys
- Backend configuration
